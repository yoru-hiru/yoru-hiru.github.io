    版本：8.3.3

    安装

    npm install -g framework7-cli

    使用

    创建项目

    framework7 create --ui # 或 framework7 create
    参数
    --skipUpdate跳过更新检查
    --port portNumber更改端口
    npm install

    开发服务器

    npm run dev
