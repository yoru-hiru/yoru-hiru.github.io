Refine

十月 7, 2024 黑猫

React 元框架。

用于 CRUD 密集型 Web 应用：

  • 内部工具
  • 管理面板
  • 仪表盘
  • B2B 应用
  • 表单

通过核心组件和钩子，

  • 身份验证
  • 访问控制
  • 路由
  • 网络
  • 状态管理
  • 国际化

 

  • 平台
    • Next.js
    • Remix
    • React Native
    • Electron
  • UI
    • TailwindCSS
    • Ant Design
    • Material UI
    • Mantine
    • Chakra UI
  • 验证
  • 后端

使用

npm create refine-app@latest
  1. 选择模板
    • Vite
    • Next
  2. 选择后端服务
  3. 选择 UI 框架
